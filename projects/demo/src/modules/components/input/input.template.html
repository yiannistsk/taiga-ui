Input — is a singleline textfield. Not recommended to use with numbers, because the value of this components is a string

Inputs for other types of data:

  • TextArea — for multiline text
  • InputDate — for dates
  • InputDateRange — for a range of dates
  • InputNumber — for number (with measurement postfix)
  • InputPassword — for passwords
  • InputPhone — for phone numbers
  • InputTag — for inputing tags
  • InputRange, InputSlider — for accurate number values
  • Slider, Range — for not accurate number values
  • InputCount — for integer number of smth
