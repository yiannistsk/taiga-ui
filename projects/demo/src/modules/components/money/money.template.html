Money transforms and shows money sum. There are some ways to show currency symbol and various decimal settings

Set singleColor to true if component is used in a text

Shows positive number with green color and negative numbers with red Currency
  1. not-zero — shows not zero pennies (by default)
  2. always — always shows pennies
  3. never — never shows pennies (do not show them without rounding)
A number of digits after comma
  1. negative-only — shows sign for negative only (by default)
  2. always — always shows sign (except 0)
  3. never — never shows sign
  4. force-negative — always shows minus (except 0)
  5. force-positive — always shows plus (except 0)
