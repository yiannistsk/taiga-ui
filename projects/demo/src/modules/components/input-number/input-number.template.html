InputNumber — is a component to inout numbers. Inputed value is formatted: thousands are separated with space. A control value of the component is a number.

Component can be used to input money or sum of something if you choose a currency symbol as a postfix (see tuiCurrency pipe samples).

There are also other components to input numbers: InputCount for integers under 999, Slider and InputSlider for inputing with a slider
