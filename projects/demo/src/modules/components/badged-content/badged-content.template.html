BadgedContent is a wrapper for other components to add badges and notifications to them.

Bottom right content. If it is a number, badge shows it. If it is a string, it will be used as icon. Warning: use big icons for all sizes except 'xs'. Top right content. If it is a number, badge shows it. If it is a string, it will be used as icon. Warning: use big icons for all sizes except 'xs'. Bottom right content color. If there is no content, shows a small circle of this color. Top right content color. If there is no content, shows a small circle of this color. Size Rounded mode
