Directive allows to customize TuiPrimitiveTextfield with parameters that it adds into DI tree

That means that you can use the directive on any root element and it will be customize all textfields in this scope

If there are several controllers above textfield, their values will be merged. The nearest directives are in priority

  1. Import an Angular module for forms and TuiTextfieldControllerModule in the same module where you want to use our directive:

  2. Use it on tui-primitive-textfield or its parent elements
